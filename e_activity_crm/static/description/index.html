eActivityCRM

Automatic & Recurring Activities for CRM Leads Odoo 19

Whenever a lead is created, the system instantly schedules the activity you configured (type + local time) and, if desired, keeps cloning it at the interval you set until the lead moves out of the chosen stage.

Create lead with activity

Generates a new activity when creating the lead.

Generic

Pick activity type, due-time and for automatic activity creation.

Timezone aware

The due hour is always shown in the user’s timezone; daylight-saving is handled by Odoo’s core.

Stage-gated recurrence

A new “Allow recurring activities” checkbox is added to CRM stages – only leads in those stages can receive periodic copies.

Stage settings detail

Fine-tune which stages allow recurrence.

Immutable history

Once an activity is repeated (cloned) the original becomes read-only (repeated = True) so your audit trail is safe.

Cron driven

A lightweight cron job (ir.cron) looks every configured minutes for activities that match: active = True, res_model = 'crm.lead', is_periodic = True, repeated = False, repetition_period > 0 and duplicates them.
